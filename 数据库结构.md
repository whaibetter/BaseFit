# BaseFit 数据库结构文档

> 数据库版本: 3  
> ORM框架: Room

---

## 枚举类型

### ExerciseCategory (动作分类)

| 值 | 说明 |
|---|---|
| `BODYWEIGHT` | 自重训练 |
| `STRENGTH` | 力量训练 |
| `CARDIO` | 有氧运动 |

### MediaType (媒体类型)

| 值 | 说明 |
|---|---|
| `IMAGE` | 静态图片 |
| `GIF` | 动图 |
| `VIDEO` | 视频 |

---

## 数据表

### 1. exercises (动作表)

存储所有健身动作的基本信息。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | Long | 主键，自增 |
| `name` | String | 动作名称 |
| `category` | ExerciseCategory | 动作分类 |
| `isActive` | Boolean | 是否激活，默认 true |
| `createdAt` | Long | 创建时间戳 |

**索引**: 无额外索引

---

### 2. week_plans (周计划表)

存储每周固定日期的训练计划。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | Long | 主键，自增 |
| `exerciseId` | Long | 关联动作ID |
| `dayOfWeek` | Int | 星期几 (1-7，周一到周日) |
| `targetSets` | Int | 目标组数 |
| `targetReps` | Int | 每组目标次数 |
| `isActive` | Boolean | 是否激活，默认 true |
| `createdAt` | Long | 创建时间戳 |

**索引**: 无额外索引

---

### 3. challenge_plans (挑战计划表)

存储限时挑战计划，如"30天完成1000个俯卧撑"。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | Long | 主键，自增 |
| `exerciseId` | Long | 关联动作ID |
| `name` | String | 挑战名称 |
| `startDate` | Long | 开始日期时间戳 |
| `endDate` | Long | 结束日期时间戳 |
| `targetTotalReps` | Int | 总目标次数 |
| `targetSets` | Int | 每天建议组数 |
| `targetReps` | Int | 每组建议次数 |
| `isActive` | Boolean | 是否激活，默认 true |
| `createdAt` | Long | 创建时间戳 |

**索引**: 无额外索引

---

### 4. check_ins (打卡记录表)

存储每次训练的打卡记录。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | Long | 主键，自增 |
| `exerciseId` | Long | 关联动作ID |
| `date` | Long | 打卡日期时间戳（只保留日期部分） |
| `completedSets` | Int | 完成组数 |
| `completedReps` | Int | 完成次数 |
| `weight` | Float? | 重量（可选，力量训练用） |
| `durationMinutes` | Int? | 时长分钟数（可选，有氧运动用） |
| `notes` | String? | 备注 |
| `createdAt` | Long | 创建时间戳 |

**索引**: 无额外索引

---

### 5. exercise_media (动作媒体资源表)

存储动作关联的图片、GIF、视频等媒体资源。

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | Long | 主键，自增 |
| `exerciseId` | Long | 关联动作ID（外键，级联删除） |
| `type` | MediaType | 媒体类型 |
| `fileName` | String | 文件名 |
| `localPath` | String? | 本地存储路径 |
| `remoteUrl` | String? | 远程URL（未来云存储扩展） |
| `thumbnailPath` | String? | 缩略图路径（视频用） |
| `orderIndex` | Int | 排序索引，默认 0 |
| `description` | String? | 媒体描述 |
| `createdAt` | Long | 创建时间戳 |

**外键约束**: 
- `exerciseId` → `exercises.id`，删除时级联删除

**索引**: 
- `exerciseId`

## 辅助数据类 (非数据库表)

以下数据类用于统计和展示，不对应数据库表：

### Achievement (成就记录)
| 字段 | 类型 | 说明 |
|---|---|---|
| exerciseId | Long | 动作ID |
| exerciseName | String | 动作名称 |
| totalCheckIns | Int | 总打卡次数 |
| totalSets | Int | 总组数 |
| totalReps | Int | 总次数 |
| maxWeight | Float? | 最大重量 |
| maxDuration | Int? | 最长时长 |
| currentStreak | Int | 当前连续天数 |
| bestStreak | Int | 最佳连续天数 |

### DailyStats (日统计)
| 字段 | 类型 | 说明 |
|---|---|---|
| date | Long | 日期 |
| totalExercises | Int | 计划动作数 |
| completedExercises | Int | 完成动作数 |
| totalSets | Int | 总组数 |
| totalReps | Int | 总次数 |

### WeeklyStats (周统计)
| 字段 | 类型 | 说明 |
|---|---|---|
| weekStart | Long | 周开始日期 |
| totalDays | Int | 总天数 |
| activeDays | Int | 活跃天数 |
| totalCheckIns | Int | 总打卡次数 |
| totalSets | Int | 总组数 |
| totalReps | Int | 总次数 |

### MonthlyStats (月统计)
| 字段 | 类型 | 说明 |
|---|---|---|
| month | Int | 月份 (1-12) |
| year | Int | 年份 |
| totalDays | Int | 总天数 |
| activeDays | Int | 活跃天数 |
| totalCheckIns | Int | 总打卡次数 |
| totalSets | Int | 总组数 |
| totalReps | Int | 总次数 |
