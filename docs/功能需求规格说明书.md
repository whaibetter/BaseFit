## 1. 项目概述

### 1.1 项目定位

BaseFit 是一款面向个人用户的轻量级健身计划管理与打卡记录工具，采用现代化的 Android 技术栈构建，帮助用户制定健身计划、记录完成情况、追踪健身进度。

### 1.2 目标用户

| 用户特征 | 描述 |
|---------|------|
| 主要用户 | 个人用户（仅自己使用） |
| 用户特点 | 追求简单高效，不需要复杂的社交功能 |
| 使用场景 | 日常健身记录、计划制定、进度追踪 |

### 1.3 目标平台

- **平台**：Android 手机原生应用
- **最低 SDK**：API 24（Android 7.0）
- **目标 SDK**：API 34（Android 14）

### 1.4 技术选型

| 组件 | 技术选型 |
|------|---------|
| 编程语言 | Kotlin 1.9.20 |
| UI 框架 | Jetpack Compose |
| 设计系统 | Material Design 3 |
| 架构模式 | MVVM + Repository Pattern |
| 数据库 | Room 2.6.1 |
| 导航 | Navigation Compose 2.7.6 |
| 图表库 | Vico Charts 1.2.0 |
| 构建工具 | Gradle 8.2.0 |

| 功能编号 | 功能描述 | 输入 | 输出 | 处理逻辑 |
|---------|---------|------|------|---------|
| F-M3.1-001 | 创建周计划 | 动作ID、星期、目标组数、目标次数 | 创建周计划记录 | 插入 week_plans 表 |
| F-M3.1-002 | 查看周计划列表 | 无 | 按星期分组显示所有周计划 | 关联 exercises 表查询 |

**数据约束**：

- `targetSets`：>= 1
- `targetReps`：>= 1

#### M3.2 挑战计划

| 功能编号 | 功能描述 | 输入 | 输出 | 处理逻辑 |
|---------|---------|------|------|---------|
| F-M3.2-001 | 创建挑战计划 | 动作ID、挑战名称、开始/结束日期、目标次数、每天组数/次数 | 创建挑战记录 | 插入 challenge_plans 表 |
| F-M3.2-002 | 查看进行中挑战 | 无 | 当前时间范围内的挑战 | 过滤 isActive=true 且当前时间在范围内 |
| F-M3.2-003 | 查看挑战进度 | 挑战ID | 已完成次数/总目标次数 | 统计 check_ins 表中该动作在期间内的打卡 |

**挑战计划数据模型**：

| 字段 | 类型 | 说明 | 约束 |
|------|------|------|------|
| name | String | 挑战名称，如"30天1000次俯卧撑" | 非空 |
| startDate | Long | 开始日期时间戳 | 必须 |
| endDate | Long | 结束日期时间戳 | 必须 > startDate |
| targetTotalReps | Int | 总目标次数 | >= 1 |
| targetSets | Int | 每天建议组数 | >= 1 |
| targetReps | Int | 每组建议次数 | >= 1 |

## 4. 用户操作流程

### 4.1 首次使用流程

```
用户启动应用
    ↓
检查是否有数据（isInitial=true 且 plans.isEmpty() && exercises.isEmpty()）
    ↓
是 → 初始化演示数据（DemoData.initializeDemoData）
    ↓
显示首页（HomeScreen）
```

### 4.2 创建周计划流程

```
用户进入"计划"页面
    ↓
点击"添加周计划"
    ↓
选择动作（从动作列表选择或新建）
    ↓
选择星期（1-7）
    ↓
设置目标组数和次数
    ↓
点击保存
    ↓
调用 PlanViewModel.addWeekPlan()
    ↓
返回计划页面刷新列表
```

### 4.3 打卡流程

```
用户进入首页或点击计划项
    ↓
查看今日计划及完成状态
    ↓
点击"打卡"按钮
    ↓
选择快速打卡（使用预设目标值）或详细打卡（填写实际数据）
    ↓
点击保存
    ↓
调用 CheckInViewModel.saveCheckIn() 或 HomeViewModel.quickCheckIn()
    ↓
返回首页刷新状态
```

### 4.4 查看统计流程

```
用户进入"统计"页面
    ↓
加载成就列表、里程碑、趋势图等
    ↓
查看本周/本月训练统计
    ↓
查看分类分布饼图
    ↓
切换周/月趋势图表
```
